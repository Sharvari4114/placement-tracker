# placement-tracker
Placement Tracker is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js) to help students monitor and improve their placement preparation. The application enables users to record their daily practice topics, track mistakes, analyze weak areas, and visualize their learning progress through an interactive dashboard.

The system allows students to add practice entries by specifying the topic, completion status, mistake category, and time spent. Based on the stored data, it automatically generates insightful analytics, including topic-wise distribution, weak topic identification, strong topic recognition, progress statistics, and learning streaks. Interactive charts provide a clear overview of study patterns, helping students focus on areas that need improvement.

✨ Features
📚 Add and manage daily study/practice entries
📊 Interactive dashboard with topic-wise analytics
🥧 Pie chart visualization of topic distribution
⚠️ Automatic identification of weak topics
✅ Highlights strong topics based on performance
🎯 Tracks preparation goals and daily streaks
⏱️ Records time spent on each topic
📝 Stores mistake types for targeted improvement
📱 Responsive and modern React-based user interface
💾 MongoDB database integration for persistent storage
⚡ REST API built with Express.js and Node.js
